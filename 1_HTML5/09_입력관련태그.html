text 관련 input 태그

        ** text 관련 type의 공통 송석 **
        - size : 입력 상자의 크기를 지정하는 속성
        - maxlength : 입력 가능한 문자열의 최대 길이
        
        - placeholder : 입력 상자에 작성될 내용 설명
            (text 뿐만 아니라 number type에서 사용 가능!)

type="text"

        - 한 줄 짜리 문자열을 입력할 수 있는 입력 상자
        - input 태그의 기본값(type을 작성하지 않으면 type="text")

        ** 모든 input 태그의 형식은 inline 형식이다. **
